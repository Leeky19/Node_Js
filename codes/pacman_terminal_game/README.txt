PAC-MAN terminal game in NodeJs 

Developed by Julien ARZOUR---COTTET

(the score and the transformation of the map are missing)

To start the game, enter the following command in the terminal:

node .\pacman.js



!!!!!! WARNING !!!!!!

First step install the blessed module,
if you don't have it here is a command to enter before launching the game:
 
npm install blessed
